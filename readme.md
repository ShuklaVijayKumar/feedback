heroku deployment

verify version
>heroku -v

install if not available from web
not login on heroku on your machine
>heroku login

create heroku app
>heroku create

you'll get one git repository name, add repository.
> git remote add heroku 'YOUR GIT LOCATION IN LAST COMMAND'

push your master 
>git push heroku master 

open the app
>heroku open

check logs
>heroku logs
